This folder contains the spectral model of the AGN NGC 1068. 

The columns in each file are: Format <Energy in keV> <Spectrum in ph cm^-2 s^-1 keV^-1>
The energy band used is 100 keV to 10000 keV


The baseline model is a powerlaw with exponential cut-off. The model paper, Bauer+2015.


NGC_1068_ec200_COSI.dat - Gamma=1.92, Ecut=200 keV; intrinsic flux 2-10 keV = 8.9e-10 erg/cm2/s

The goal of these data for COSI is: determine flux in the COSI band, and coronal cut-off location
